# 🎨 CodeCanvas – Resume Builder Web App

🌐 **Live Demo:** https://codecanvas-3.onrender.com
📂 **GitHub Repo:** https://github.com/amanversee/CodeCanvas

CodeCanvas is a modern and customizable **Resume Builder Web Application** built with **React** and **TailwindCSS**.
It allows users to create, edit, and manage resumes with multiple professional templates in a clean and intuitive interface.

---

## 🚀 Features

* 🧾 Create new resumes
* 📂 Upload existing resumes (PDF)
* 🎨 Multiple resume templates
* ✏️ Edit resume title and content
* 🗂 Dashboard to manage all resumes
* 📱 Fully responsive design
* ⚡ Fast and lightweight UI (Vite + React)
* 🎯 Template-based resume customization

## 🧠 How It Works

1. User logs in and lands on the **Dashboard**
2. User can:

   * Create a new resume
   * Upload an existing resume
   * Select a template
3. After selecting, user is redirected to **Resume Builder**
4. Resume is rendered dynamically based on selected template
5. User can customize content and design
